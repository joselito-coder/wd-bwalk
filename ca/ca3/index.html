Some of the best Linux distros

Manjaro linux

Manjaro is a free and open-source Linux distribution based on the Arch Linux operating system. Manjaro has a focus on user-friendliness and accessibility. some people call it ARchBTW

ARch linux

Arch Linux is a Linux distribution created for computers with x86-64 processors. Arch Linux adheres to the KISS principle ("Keep It Simple, Stupid"). some people call it ARchBTW

Void linux

Void is a general purpose operating system, based on the monolithic Linux kernel. Its package system allows you to quickly install, update and remove software

Debian

Debian also known as Debian GNU/Linux, is a GNU/Linux distribution composed of free and open-source software, developed by the community-supported Debian Project, which was established by Ian Murdock on August 16, 1993.

Gentoo

Gentoo Linux is a Linux distribution built using the Portage package management system.the source code is compiled locally according to the user's preferences and is often optimized for the specific type of computer
